# Инструкция для заказчика: Как скачать и запустить сайт

Эта инструкция предназначена для того, чтобы скачать уже готовый сайт с GitHub и запустить его на своем компьютере.

## Шаг 1: Установка Git и Python

Перед началом убедитесь, что у вас установлены:
1.  **Git**: Скачать с [git-scm.com](https://git-scm.com/downloads)
2.  **Python**: Скачать с [python.org](https://www.python.org/downloads/) (версия 3.10 или новее). При установке обязательно поставьте галочку **"Add Python to PATH"**.

## Шаг 2: Скачивание проекта (Клонирование)

1.  Откройте терминал (PowerShell или Git Bash).
2.  Перейдите в папку, куда хотите скачать сайт (например, на рабочий стол):
    ```bash
    cd Desktop
    ```
3.  Выполните команду клонирования репозитория:
    ```bash
    git clone https://github.com/BeReZin-DaNiL/site_for_photograf
    ```
4.  После скачивания зайдите в папку проекта:
    ```bash
    cd site_for_photograf
    ```

> **Важно:** Не используйте команды `git init`, `git add` или `git push` — они нужны для *создания* нового проекта, а вы *скачиваете* уже существующий.

## Вариант 2: Если вы хотите загрузить этот код в СВОЙ GitHub

Если вы уже скачали файлы, внесли изменения и хотите выложить их в **свой** репозиторий (а не просто скачать наш), выполните эти действия.

**Ошибка на вашем скриншоте (`fatal: not a git repository`) означает, что в папке не была выполнена команда `git init`.**

### Инструкция:

1.  Откройте терминал в папке с проектом (где лежат ваши файлы).
2.  Инициализируйте Git (превратите папку в репозиторий):
    ```bash
    git init
    ```
3.  Добавьте все файлы:
    ```bash
    git add .
    ```
4.  Сохраните изменения (сделайте коммит):
    ```bash
    git commit -m "Моя версия сайта"
    ```
5.  Переименуйте ветку в `main`:
    ```bash
    git branch -M main
    ```
6.  Привяжите папку к вашему репозиторию на GitHub:
    *   Зайдите на GitHub, создайте новый пустой репозиторий.
    *   Скопируйте его ссылку (она выглядит как `https://github.com/ВАШ_ЛОГИН/НАЗВАНИЕ.git`).
    *   Вставьте ссылку в команду (замените `ССЫЛКА_НА_ВАШ_РЕПОЗИТОРИЙ`):
    ```bash
    git remote add origin ССЫЛКА_НА_ВАШ_РЕПОЗИТОРИЙ
    ```
    *(Если появится ошибка `remote origin already exists`, напишите `git remote set-url origin ССЫЛКА_НА_ВАШ_РЕПОЗИТОРИЙ`)*

7.  Отправьте файлы на GitHub:
    ```bash
    git push -u origin main
    ```
    *(Вас может попросить ввести логин и пароль/токен от GitHub)*


## Шаг 3: Настройка окружения и запуск

Находясь в папке проекта (где лежит файл `manage.py`), выполните следующие команды по очереди:

1.  **Создайте виртуальное окружение:**
    ```bash
    python -m venv venv
    ```

2.  **Активируйте его:**
    *   Windows:
        ```bash
        venv\Scripts\activate
        ```
    *   Mac/Linux:
        ```bash
        source venv/bin/activate
        ```
    *(Если все прошло успешно, в начале строки появится `(venv)`)*

3.  **Установите библиотеки:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите сайт:**
    ```bash
    python manage.py runserver
    ```

Теперь откройте браузер и перейдите по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

---
**Если нужно обновить сайт (получить новые изменения от разработчика):**
Просто откройте терминал в папке проекта и напишите:
```bash
git pull
```
